
# Unnamed Survival Computer

## What is it?
A wrist-mounted computer for survival. Using KivyPie to create a Python GUI on a Linux OS.


## Future Improvements
- Compass peripherals
- Water purity tester
- Temperature
- Wind Speed
- GPS
